# ESP32S3-NEXTION-EXAMPLE

This directory contains example projects for the ESP32-S3 development board (ONX3248G035), covering peripherals such as LCD touch, camera, audio, SD card, Wi-Fi, and battery. For details of each example, see the README in the corresponding example folder.
